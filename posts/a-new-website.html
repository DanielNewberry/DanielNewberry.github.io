A new website

2024-06-07

Hello!

This is my 1st 2nd 3rd 4th 5th 6th 7th 8th 9th attempt at a personal website. I've had many websites before, but they didn't do what I wanted them to do (mainly because I didn't take the time to learn how to get them to do what I wanted them to do).

Previously I've used all-in-one website builders and hosting such as Squarespace and Weebly, the Canva Websites builder that comes with the Canva Pro membership and website builder frameworks such as Hugo (built on the Go Language).

Whilst they all had their merits, there were too many issues for me to deal with.

Squarespace looked great and had a lot of dynamic responsiveness built in. But it was expensive and there were so many barriers to posting content such as navigating menues and waiting for elements to load in. I realised it was great for dynamic websites, which I simply didn't need (or want).

Weebly was a nightmare. Poor mobile optimisation and the drag-and-drop was limited.

Canva Websites was good for teh ease of setting it up, however it would be expensive if I wasn't already paying for the Canva Pro membership. Again, mobile optimisation was poor - arguably the worst out of every website build I have used. And there were no blogging or posting features so anything outside of a one-page website was impossible without creating multiple designs.

The Hugo framework was good, but there were a lot of learning curves. Also, I realised that when compiling the website to update, it was just updating and creating new HTML and CSS files.

Hugo got me closest to how I wanted the website to look and work, especially when running the no-style-please! theme.

I'm a big fan of more minimal looking websites where information is front and center. When I found the new.css stylesheet I realised that I would be able to design the exact website that I wanted.

So I have - I'm just editing changes directly as HTML in Sublime Text and pushing changes to a Github Pages repository. This lets me use a custom domain and as long as the repository is kept as open source, hosts the website for free too.

Perfect.

This website is built using plain HTML with no active content. It is designed to load instantaneously on any platform, even at low download speeds.
